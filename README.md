# Vehicle System for Roblox

Готовая система управления машинами для гонок и других игр. Простое управление, настраиваемая скорость, камера от третьего лица.

## Возможности

- Управление машиной с клавиатуры (WASD или стрелки)
- Настраиваемая скорость и поворот
- Камера от третьего лица
- Посадка в машину по нажатию E
- Автоматическое притягивание игрока к сиденью

## Как установить

1. Скачай CarController.lua
2. Помести его в модель машины (внутрь модели автомобиля)
3. Убедись что у машины есть:
   - PrimaryPart (главная часть)
   - Seat (сиденье) с прикреплённым Weld
   - HumanoidRootPart у игрока
4. Настрой параметры под свою машину

## Настройка

В начале скрипта:

local MAX_SPEED = 100            -- максимальная скорость
local ACCELERATION = 30          -- ускорение
local TURN_SPEED = 3             -- скорость поворота
local BRAKE_FORCE = 50           -- сила торможения

## Управление

- W / Стрелка вверх — газ
- S / Стрелка вниз — тормоз/задний ход
- A / Стрелка влево — поворот налево
- D / Стрелка вправо — поворот направо
- E — сесть в машину / выйти

## Как это работает

1. Игрок нажимает E рядом с машиной
2. Персонаж притягивается к сиденью
3. Управление переключается на машину
4. При повторном нажатии E — игрок выходит

## Подходит для

- Гонок
- РП игр
- Симуляторов вождения
- Любых игр с транспортом
